Download files by dragging and dropping them onto the web page or by using the 'Add' button in the top right-hand corner. Note that only references to the files are downloaded at this time: the file contents are downloaded only when they are played.

Click the 'Start' button to start the playback. You can also start playing by clicking on a name in the list. You can change the order of playback by dragging and dropping names within the list. The 'Shuffle' button will rearrange your list into a random order.

Troubleshooting If playback stops after the first song, check that you have not disabled 'auto-play' in your browser.
